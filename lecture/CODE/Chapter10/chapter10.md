[논리와 스위치]
===================================================================================================================
* 부울 대수
```
조지 부울은 영국의 수학자이자 논리학자이며, 논리대수인 부울 대수를 창안해 기호논리학 분야에 큰 업적을 남긴 인물입니다. 그가 창안한 대울 대수는 어떤
명제의 참과 거짓을 이진수 1과 0에 대응시켜서 명제와 명제간의 관계를 수학적으로 표현하는 것입니다. 부울 대수에서 + 기호는 두 종류의 합집합(union)을
의미합니다. 두 종류의 합집합이란 첫 번째 종류나 두 번째 종류에 속하는 것을 모두 포함하는 것을 의미합니다. 또 × 기호는 두 종류의 교집합(intersection)
을 의미합니다. 두 종류의 교집합은 첫 번째 종류에 속하는 동시에 두 번째 종류에도 속하는 것들을 의미합니다. 그리고 '1'이라는 기호는 전부를 나타내고, 
'0'은 아무것도 포함하지 않는 종류를 의미합니다. 전체집합과 공집합이라고 할 수 있습니다.
```

* AND 회로
```
:그림1:과 같이 회로를 만든다고 하면 두 개의 스위치 모두를 닫아야 전구에 불이 들어옵니다. 덧붙이자면 두 스위치 모두 닫힌 경우를 제외하고 불이 켜지지
않습니다. 두 개의 스위치가 집합이라고 가정했을때 두 집합이 모두 만족해야(닫혀야) 전구에 불이 켜지므로 AND 회로는 교집합이라고 할 수 있으며, 부울 대수
로는 × 기호로 나타낼 수 있습니다.  
```

* OR 회로
```
:그림2:와 같이 회로를 만든다고 하면 어느 하나의 스위치만 닫혀도 전구에 불이 들어옵니다. 덧붙이자면 두 스위치 모두 열린 경우를 제외하고 모두 불이
켜집니다. 두 개의 스위치가 집합이라고 가정했을때 두 집합중 어느 하나만 만족해도(닫혀도) 전구에 불이 켜지므로 OR 회로는 합집합이라고 할 수 있으며,
부울 대수로는 + 기호로 나타낼 수 있습니다. 
```

* 복잡한 질문
```
애완동물 가게에서 판매원에게 "중성화 된 수컷 고양이인데, 하얀색이나 황갈색이었으면 좋겠어요. 아니면 중성화된 암컷 고양이에 하얀색만 아니면 괜찮을 것
같아요. 음... 검은 고양이면 어떤 고양이든 상관없을 것 같긴하네요."라고 물어본다면 판매원은 너무도 당황스러울것 같습니다. 이 질문에는 어려운 점은 없지만
단순한 가능성들이 모여서 복잡함을 만든 것이라 볼 수 있습니다. 판매원에게 :그림3:과 같은 표가 있다면 고객의 질문은 몇 가지 스위치로 알아볼 수 있습니다.
:그림3:의 표로 질문을 해석하면 부울 대수에 기반한 '(M × N ×(W + T)) + F × N × (1 - W)) + B'의 식을 도출해볼 수 있습니다. 또 이 식으로
회로를 만든다면 :그림4:와 같은 회로를 만들어 볼 수 있습니다. 이제 판매원은 고양이에 대한 스위치를 눌려가며 고객이 원하는 고양이인지를 전구를 통해 확인
할 수 있게 되었습니다.
```
