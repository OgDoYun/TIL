모스부호를 배웠으니 이제 친구와는 손전등의 움직임이 아닌 켜고 껴짐으로 대화할 수 있게 되었고,
손전등을 분해해보면서 전기라는 것에 대해 조금은 이해한 것같습니다. 하지만 손전등의 빛이 항상 일정하게 밝지도 않고,
지나다니는 사람에 의해 불빛이 가려지기라도 한다면 모스부호가 엉키게 됩니다.
손전등의 전구부분을 친구집으로 옮겨줄만큼 매우 큰 손전등을 만드려면 어떡해야 할까요?

```

* 공통 연결

공통 연결을 통해 전선의 길이를 25% 정도 줄였습니다.
그림 첨부
```
* 좋은 전도체
```
손전등을 만들 수 있게되었지만 전선을 잇는데 엄청난 돈이 들어간다는 것을 알았습니다. 공통부분을 대신할만한 전도체는 바로 지구입니다.
지구는 대부분이 무생물인 금속, 돌, 물, 그리고 유기물로 이루어진 지름 약 12000킬로미터에 달하는 거대한 구형 물체입니다.
앞 선 설명에서 좋은 전도체에 대해 이야기할 때 구리, 은, 금에 대해서는 언급했지만, 자갈이나 뿌리와 같은 건 이야기한 적 없습니다.
사실 지구를 구성하는 여러 가지 물질 중에 축축한 지면과 같은 일부 구성요소는 건조한 모래 같은 것들보다는 더 좋은 전도체이지만,
전반적으로 봤을 때 그다지 아주 좋은 도체라 할 수 없습니다. 하지만, 앞에서 전도체의 특성 중 하나를 배웠습니다.
바로 '크면 클수록 좋다'입니다. 매우 굵은 전선은 매우 얇은 전선보다 훨씬 더 전도성이 좋다는 것입니다. 
이 부분에서 지구는 그 어떤 전도체보다 굵다고 할 수 있습니다. 지구를 전도체로 사용하기 위해 넓은 표면적을 가지고 있는 도체를
이용하여 지구와 많은 양의 접촉면을 만들고 이를 유지해주어야 합니다. 이렇게 지표면에 전기적 접점을 두는 것을 접지라고 하며,
영국에서는 어스(earth), 미국에서는 그라운드(ground)라고 부릅니다. 이를 통해 전선의 길이를 50% 정도 줄였습니다.
```
