[전신과 릴레이]
======================
* 전신(telegraph)이 나오기 전 시대적 배경
```
1800년 초기에는 장거리 통신을 편지로 주고 받았기 때문에 우편마차, 열차 혹은 배를 이용하였기에 배달되는 시간이 길었습니다.
모스의 발명 이전에도 수십 년 동안 장거리 통신의 속도를 높이고자 하는 다양한 시도가 있었습니다. 언덕마다 사람이 깃발을 들고
서서 신호가 오는 것을 기다리며 그 신호를 중계했습니다. 그렇다고 다른 시도가 없었던 것은 아닙니다. 전신의 개념 자체는
1800년대 초반에 이미 널리 퍼져 있었다는 것은 확실합니다. '전선으로 연결된 한쪽 끝에서 어떤 일을 하면 그것이 전선의
맞은 편에서 어떤 일을 유발시킬 수 있다'라는 전기의 원리를 이용하는 전신 시스템의 개념은 기본적으로는 아주 간단합니다. 
```
* 전신의 원리
```
그렇다고 모스가 전기적인 특성만으로 전신 시스템을 구축했다면 전구를 사용했을텐데 아쉽게도 전구의 발명은 1879년입니다.
모스는 그 대신 전자기(electromagnetism)현상을 이용했습니다. 강철막대기에 얇은 전선을 수백번 감고 전선에 전류를 흘리면, 
강철 막대기가 마치 자석처럼 동작합니다. 이렇게 되면 강철 막대기에는 다른 강철이나 쇠에 달라붙게 됩니다. 그리고 전류가 흐르지
않으면 강철 막대기는 자성을 잃어버리게 됩니다. 이를 전자석이라고 하는데 이것이 전신 시스템의 기반이 되었습니다. 이 전자석을
이용하여 스위치를 닫으면(누르면) 전류가 흘러 전자석이 강철 막대를 자성으로 끌어당겨 딸깍하며 소리를 냅니다. 또 스위치를 열면
(때면) 전류가 흐리지 않아 전자석이 자성을 잃어 강철 막대를 다시 놓아주고 강철 막대를 같이 붙은 스프링에 의해 제자리로 돌아가
면서 딱!하고 소리를 냅니다. 이를 통해 딸깍..딱!하면서 한쪽 끝에서 보낸 스위치의 움직임이 다른 한쪽 끝의 강철 막대의 움직임을
만들면서 소리도 내게 되었습니다. 이제 스위치를 짧게 눌러 점을, 길게 눌러 선을 표시하면 먼 곳에서도 모스부호를 보낼 수 있게
되었습니다. 
```
* 릴레이의 탄생
```
하지만 전신국들은 큰 고민에 빠졌습니다. 전신을 통해 좋은 의사소통 수단을 얻었지만, 전신을 위한 전선이 길면 길어질수록 전선의
저항이 높아졌고 전압을 계속 높일 수 없었기에 의사소통의 물리적 한계에 부딪쳤습니다. 가장 쉬운 해결책은 전압을 높일 수 없으니
전신을 중간에 놓고 중계를 하는 방법이었습니다. 이 또한 처음엔 사람이 직접 듣고 모스부호를 해독한 뒤 다시 스위치를 조작했지만
나중에는 신호를 받은 강철막대의 움직임을 그대로 신호를 보내야할 스위치에 전달하면서 자동으로 이루어졌습니다.
이를 중계기(repeater)라고 부르거나 릴레이(relay)라고 불렀습니다.
```
